# 프로젝트 설정 및 실행 방법

## 설치 및 실행

1. **프로젝트 클론**

```bash
git clone https://github.com/eunoia-jason/react_base_file.git
```
2.**개발 환경 설정**
   ```sh
   npm install
   ```
   ```sh
   npm start
   ```
